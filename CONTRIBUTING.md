# Contributing to Awesome OpenSpec

Thank you for your interest in contributing!

## How to Contribute

1. Fork this repository
2. Add your resource to the appropriate section in `README.md`
3. Submit a pull request

## Guidelines

- Make sure the resource is related to OpenSpec or Spec-Driven Development
- Use the format: `[Name](URL) - Short description.`
- Keep the entire line under 150 characters so it renders on one line on GitHub
- Keep descriptions concise (one sentence)
- Add new entries in alphabetical order within the relevant section
- Check that links are working

## What to Add

We welcome:

- Tools and utilities that work with OpenSpec
- Tutorials, articles, and blog posts
- Videos and presentations
- Schemas and extensions
- GitHub Actions and integrations
- Community resources

## Questions?

Feel free to open an issue if you're unsure about anything.
